Why this site?

A simple web package can find openQA jobs/jobgroups which use specific test module.

We have many test modules which may be used by other job groups.

If we do some code changes, we need make sure VRs should be done for all these jobs.

It will be an easy way to find all these jobs.

How to use it?

Step 1:
Input the test module you wanna to search

Step 2:
Click the button search

Step 3:
Will jump another page to show the result
